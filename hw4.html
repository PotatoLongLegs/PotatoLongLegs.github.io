List Usage
Unordered List A bullet point list with no specific order. On webpages, they use it to list items that do not need to be in any specfic order like a list of books. To start the unordered list use this tag < ul >
Ordered List A numbered list. On webpages, they use it to list items in a specific order like baking a cake. To start the ordered list use this tag < ol >
Definition List A term and a definition, always in a pair. On webpages, they use it as a dictionary. To start the definition list use this tag < dl >
